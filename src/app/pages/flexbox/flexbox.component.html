Propiedades del contenedor
flex-direction
row row-reverse column column-reverse
Define la dirección en la que se colocan los elementos hijos dentro de un contenedor flexbox. Puede tomar uno de estos valores
justify-content
flex-start flex-end center space-between space-evenly
Sirve para alinear los elementos hijos a lo largo del eje principal de un contenedor flex,permite controlar la distribución del espacio entre y alrededor de los elementos flexibles dentro del contenedor, facilitando así un diseño más dinámico y flexible.
align-items
stretch flex-start flex-end center baseline
Sirve para alinear los elementos hijos a lo largo del eje transversal de un contenedor flex. permite controlar la alineación y distribución de los elementos flexibles a lo largo del eje transversal, proporcionando una manera efectiva de manejar el espacio vertical u horizontal entre los elementos.
flex-wrap
nowrap wrap wrap-reverse
Sirve para controlar cómo los elementos flexibles se envuelven en múltiples líneas dentro de un contenedor flex. Esta propiedad es útil cuando hay más elementos flexibles de los que el contenedor puede mostrar en una sola línea, permitiendo que los elementos se distribuyan en varias líneas o columnas según sea necesario
align-content
stretch flex-start flex-end center space-between space-around
Sirve para alinear y distribuir el espacio entre las líneas de elementos flexibles a lo largo del eje transversal de un contenedor flex. Esta propiedad es relevante cuando hay múltiples líneas de elementos dentro del contenedor, lo cual ocurre cuando se utiliza flex-wrap con el valor wrap o wrap-reverse.
Propieades del item
flex-grow
Define cuánto puede crecer ese ítem en proporción al espacio disponible en el contenedor, en comparación con otros ítems. Su valor por defecto es 0, lo que significa que el ítem no crecerá más allá de su tamaño inicial. Si se establece un valor mayor a 0, el ítem crecerá en función del espacio sobrante: un valor de 1 indica que el ítem tomará una parte proporcional del espacio disponible, y valores superiores harán que el ítem crezca más en relación a los demás ítems.
flex-shrink
Controla la capacidad de un ítem dentro de un contenedor flex para reducir su tamaño cuando el espacio es limitado. Por defecto, tiene un valor de 1, lo que permite a los ítems encogerse si es necesario. Un valor de 0 evita que el ítem se reduzca, mientras que valores mayores permiten que el ítem se reduzca más en comparación con otros ítems. Esto ayuda a distribuir el espacio de manera eficiente cuando el contenedor es más pequeño que la suma de los tamaños iniciales de los ítems.
flex-basis
Define el tamaño inicial de un ítem dentro de un contenedor flex antes de distribuir el espacio restante. Por defecto, su valor es auto, lo que hace que el tamaño del ítem dependa de su contenido o de las propiedades width y height. Puedes especificar un tamaño fijo con unidades como px o %, lo que establece una base específica antes de que se apliquen las propiedades flex-grow y flex-shrink. Si se establece en 0, el tamaño inicial del ítem será nulo y dependerá completamente del comportamiento flexible para determinar su tamaño final.
align-self
auto stretch flex-start flex-end center baseline
Permite ajustar la alineación vertical de un ítem individual dentro de un contenedor flex, sobrescribiendo el comportamiento establecido por align-items alineando el ítem de manera diferente en el eje transversal. El valor por defecto es auto, lo que hace que el ítem siga la alineación definida por align-items. Esta propiedad es especialmente útil para alinear un ítem de manera distinta al resto en el mismo contenedor flex.
order
Permite controlar el orden visual de los ítems dentro de un contenedor flex, independientemente de su orden en el código HTML. Su valor por defecto es 0, pero al asignar números enteros, ya sean positivos o negativos, se puede cambiar la posición de los ítems: los valores más bajos se muestran primero, y los más altos después.
